# How to Contribute

## Start

```shell
yarn
# the command will install project root dependencies and demos/react dependencies
```

## Develop

* test it by `vitest`
```shell
yarn test
```

* preview develop(by react).
```shell
cd demos/react
yarn dev
```

* edit it in your editor.

## End

* commit your changes.
* push your changes.
* create a pull request.
* wait for review.
* merge it.

That's all. Thank you for your contribution.
